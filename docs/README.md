## Démarrage rapide

1. Connectez-vous sur `/login.html`
2. Allez dans **Séances** > **Nouvelle séance**
3. Ajoutez résolutions et configurez les politiques
4. Passez en mode **Live** pour démarrer les votes

Besoin d'aide ? Consultez la [FAQ](FAQ.md) ou le [guide fonctionnel](GUIDE_FONCTIONNEL.md).
