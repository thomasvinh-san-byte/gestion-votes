## Tech Debt Restant

**Pas de blocker. Items polish post-MVP reportes a v1.3 :**

1. **Browser test execution sur le host** — phpredis non installe sur la machine de dev (host) ; les tests passent dans le container Docker. Resolution : `apt install libapache2-mod-php-redis` OU utiliser le container exclusivement.
2. **2 dead settings restants** : `settMaxLoginAttempts`, `settPasswordMinLength` ont ete supprimes du HTML (Phase 11) — ils ne sont plus exposes. Si features needed : nouvelle iteration.
3. **Multi-browser** : actuellement chromium-only dans Phase 8 setup. Firefox/WebKit : milestone ulterieur.
4. **Visual regression / a11y deep audit** : axe-core baseline en place (Phase 7 v1.1) mais audit complet defere a v1.3 polish.
5. **HTMX 2.0 upgrade** : breaking changes hx-on case sensitivity, audit complet necessaire — milestone separe.

**v1.2 est shippable.**

Le projet est passe d'un etat ou :
- L'app crashait sur chaque API call (RateLimiter dead callsite)
