# v2.7 Visual Audit — Screenshots de référence

Ce répertoire reçoit les captures visuelles avant/après pour la phase 1 v2.7
(cohérence visuelle).

## Convention de nommage

`{ecran}-{etat}.png` avec `etat ∈ {avant, apres}`.

Exemples :
- `operator-live-avant.png`
- `operator-live-apres.png`
- `vote-mobile-avant.png`
- `vote-mobile-apres.png`

## Statut

Capture différée (suit le pattern Phase 1 v2.6 SC-3 deferred). Les baselines `grep`
gelées dans `.planning/v2.7-VISUAL-AUDIT.md` sont la métrique objective primaire.
Les screenshots sont produits manuellement sur dev-machine et déposés ici en clôture
de phase 1 (par 01-03 ou release v2.7).
