    Use `test.setTimeout(120000)` and `waitUntil: 'domcontentloaded'`.

    For each PRIMARY interaction, trigger AND assert an observable result:

    1. **Page mount + KPI load**
       - Navigate to `/analytics.htmx.html`

    2. **Year filter change — triggers data reload**
       - Capture current `#kpiMeetings` text.
       - `page.selectOption('#yearFilter', '2025')`
       - Wait 800ms (API reload debounce)
       - Assert `#yearFilter` has value "2025" (proves wiring).
       - Reset to "2026" and wait 800ms.

    3. **Period pills — active class toggles**
       - Click `.analytics-period-pill[data-period="30j"]`

       - Assert `.analytics-period-pill[data-period="1an"]` no longer has `.active` class.
       - Restore by clicking the `1an` pill.

    4. **Tab switch — Motions tab content becomes visible**
       - Click `.analytics-tab[data-tab="motions"]`
       - Assert `#tab-motions` becomes visible (has `.active` class or display != none) within 3s.
       - Assert `#tab-participation` is no longer active.
       - Verify donut SVG segments exist: `#donutFor`, `#donutAgainst`, `#donutAbstain` are in the DOM.

    5. **Tab switch — Timing tab**
       - Click `.analytics-tab[data-tab="timing"]`

       - Assert `#durationChart` canvas element is in the DOM.

       - Click `.analytics-tab[data-tab="anomalies"]`
       - Assert `#tab-anomalies` becomes active.
       - Assert `#anomaliesOverview` element is visible.

       - Return to Participation tab.
       - Click `#refreshBtn`.
       - Wait 1s. Assert `#kpiMeetings` still contains a digit (proves refresh did not break state).
