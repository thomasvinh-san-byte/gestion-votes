**Step 1: Add lazy ValueTranslator property (consistent with Phase 3 pattern):**
```php
private ?ValueTranslator $translator = null;

private function translator(): ValueTranslator {
    return $this->translator ??= new ValueTranslator();
}
```

**Step 2: Remove all constants, translate methods, format methods, row formatters, and header methods.**

**Step 3: Add delegation stubs for every removed public method.** Each stub is a one-liner:
```php
public function translateAttendanceMode(?string $mode): string {
    return $this->translator()->translateAttendanceMode($mode);
}
// ... same for translateDecision, translateVoteChoice, translateMeetingStatus, translateVoteSource, translateBoolean
// ... same for formatDate, formatTime, formatNumber, formatPercent
// ... same for formatAttendanceRow, formatVoteRow, formatMemberRow, formatMotionResultRow, formatProxyRow, formatAuditRow
// ... same for getAttendanceHeaders, getVotesHeaders, getMembersHeaders, getMotionResultsHeaders, getProxiesHeaders, getAuditHeaders
```

CRITICAL: formatPercent calls formatNumber internally. Since both are now on ValueTranslator, the delegation stub just calls `$this->translator()->formatPercent($value)` — the internal call happens inside ValueTranslator.

CRITICAL: The streamFullXlsx and createFullExportSpreadsheet methods call formatAttendanceRow, formatVoteRow, etc. via `$this->`. These will hit the delegation stubs on ExportService, which delegate to ValueTranslator. This is correct and preserves callable references like `[$this, 'formatAttendanceRow']`.
