**4. Wire setSseIndicator() in operator-realtime.js:**
Add the SSE_LABELS map and setSseIndicator function inside the IIFE, before the connect call:
```javascript
var SSE_LABELS = { live: '\u25cf En direct', reconnecting: '\u26a0 Reconnexion...', offline: '\u2715 Hors ligne' };
function setSseIndicator(state) {
  var el = document.getElementById('opSseIndicator');
  var lb = document.getElementById('opSseLabel');
  if (el) el.setAttribute('data-sse-state', state);
  if (lb) lb.textContent = SSE_LABELS[state] || state;
}
```
Then in the existing `onConnect` callback (line ~35), after `sseConnected = true;`, add: `setSseIndicator('live');`
In the existing `onDisconnect` callback (line ~39), after `sseConnected = false;`, add: `setSseIndicator('offline');`
Also detect reconnecting state: if EventStream has a reconnect attempt callback, use it. Otherwise, set 'reconnecting' in onDisconnect and 'offline' after a 5-second timeout if still disconnected.
