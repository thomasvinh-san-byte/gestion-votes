    **Create `public/assets/js/pages/settings.js`:**
    - IIFE pattern, use `var` throughout (codebase convention)
    - Tab switching: querySelectorAll('.settings-tab'), click handler sets active class + shows/hides panels via hidden attribute. Read URL hash on load to auto-select tab (e.g., #securite selects Securite tab).
    - Auto-save: attach change/input event listeners to all toggles, selects, and number inputs.
      - On change: debounce 500ms for text/number inputs, immediate for toggles/selects/radios.
      - Call api() (existing global function) to persist: `api('/api/v1/admin_settings.php', { action: 'update', key: fieldId, value: newValue })`.
      - On success: `AgToast.show('Parametre enregistre', 'success')` (brief toast).
      - On error: `AgToast.show('Erreur de sauvegarde', 'error')` + revert field to previous value.
      - Store previous values in a Map for revert capability.
    - Quorum policies: extract the quorum CRUD code from admin.js (lines ~397-590) — loadQuorumPolicies(), renderQuorumList(), quorum create/edit/delete with ag-confirm dialogs. Use api() for `/api/v1/admin_quorum_policies.php`.
    - Distribution keys: basic table display (already static HTML). Add "Ajouter une cle" and "Modifier" button handlers (ag-confirm modal for add/edit form).
    - Email templates: extract template edit logic — click handler on template items, show/hide templateEditor card, save/cancel buttons.
    - Text size A/A+/A++: on change, set document.documentElement.style.fontSize to '16px'/'18px'/'20px' and save to localStorage key 'ag_text_size' + API call for tenant default.
    - High contrast toggle: on change, toggle data-high-contrast attribute on documentElement, save to localStorage 'ag_high_contrast' + API call.
    - Load settings on init: call api('/api/v1/admin_settings.php', { action: 'list' }) and populate all form fields. On error, use console.warn (graceful degradation, no blocking error).
