### Shadow (cas TECH-01 BASSE confiance reportés)

| Pattern hardcodé | Token cible |
|---|---|
| `0 1px 2px rgba(0,0,0,0.05)` ou variantes 1px subtil | `var(--shadow-xs)` |
| `0 1px 2px rgb(...)` léger | `var(--shadow-sm)` |
| `0 4px 12px ...` modéré | `var(--shadow-md)` |
| `0 16px 40px ...` ample | `var(--shadow-lg)` |
| `inset 0 2px 4px ...` | `var(--shadow-inner)` |
| `inset 0 1px 2px ...` | `var(--shadow-inset-sm)` |

Si la shadow contient un fallback inline (ex: `box-shadow: 0 1px 2px var(--color-primary-rgb)`), c'est BASSE confiance — laisser et noter.

### Border (cas TECH-01 BASSE confiance reportés)

| Pattern hardcodé | Token cible |
|---|---|
| `1px solid var(--color-border)` | `var(--border-default)` |
| `1px solid var(--color-border-subtle)` | `var(--border-subtle)` |
| `1px solid var(--color-border-strong)` | `var(--border-strong)` |
| `1px dashed ...` | `var(--border-dashed)` |
| Border avec couleur sémantique (success, danger) | **garder** (pas dans le scope TECH-01) |
| Border avec width non-1px | **garder** (cas BASSE confiance, hors TECH-01) |

### Cas spéciaux **autorisés** (ne PAS remplacer)
- `padding: 0` ou `margin: 0` (resets) — garder
- `margin: auto` — garder
- Valeurs en `%` (ex: `padding: 50%` aspect ratio) — garder
- Valeurs en `em` (lié à la taille de police) — garder, documenter
- Valeurs exotiques (`7px`, `13px`, `0.4rem`) — arrondir au token le plus proche **et noter**
