# Plan 02.1 — Migrate `business_error` → 3 specific codes

## Goal

Remplacer le code générique `business_error` (baseline ~11 callers) par **3 codes spécifiques** identifiés via audit, en migrant chaque caller vers son code sémantique tout en conservant `business_error` comme fallback générique avec usage cible < 5 %.

Source : ERR-V24-01 (REQUIREMENTS.md), CONTEXT.md décisions D-01..D-04.

## Acceptance Criteria

1. **`02.1-AUDIT.md` livré** : grep complet des callers `business_error` dans `app/` + `public/api/`. Pour chaque caller : file:line, contexte (méthode), classification sémantique en 3 buckets. 3 noms de codes proposés à partir des buckets.
2. **3 codes nommés** ajoutés dans `app/Services/ErrorDictionary.php` avec messages français conformes ERR-02 (virgule + verbe d'action) et ERR-03 (0 phrase bannie). Codes existent dans le tableau `MESSAGES`.
3. **Callers migrés** : chaque caller `business_error` identifié pointe désormais vers son code spécifique. `business_error` reste dans le dictionnaire mais devient fallback rare (<5 % usage attendu).
4. **`business_error` préservé** dans le dictionnaire — pas de suppression. Message inchangé byte-stable si possible (si modifié, justifié dans SUMMARY).
5. **Test gardien permanent** : `tests/Security/ErrorDictionaryMigrationTest.php` créé, 3 tests minimum :
   - `testNewCodesExist` : 3 codes présents dans `getCodes()`
   - `testNewCodesConformErr02Err03` : virgule + verbe + 0 phrase bannie
   - `testBusinessErrorCallersBaseline` : `grep` count `'business_error'` dans `app/` + `public/api/` ≤ N (N = baseline post-migration, fixé après migration, typiquement 0-3)
6. **Régression nulle** sur tests existants : UxConventionsTest + EditorialConventionsTest + CopyConventionsTest restent verts.
7. **Baseline 206 codes byte-stable** ne s'applique plus (3 codes ajoutés → 209). PLAN documente le nouveau total.

### Task 4 — Test gardien

Créer `tests/Security/ErrorDictionaryMigrationTest.php` :

```php
<?php
declare(strict_types=1);

namespace Tests\Security;

use AgVote\Service\ErrorDictionary;
use PHPUnit\Framework\TestCase;

final class ErrorDictionaryMigrationTest extends TestCase
{
    private const NEW_CODES = ['quorum_not_met', 'meeting_state_invalid', 'vote_state_invalid'];
    private const FORBIDDEN_PATTERNS = [
        '/réessayer\\.|contactez l\'admin|erreur survenue|une erreur est survenue|veuillez réessayer plus tard/i',
    ];
    private const ACTION_VERBS = ['vérifiez', 'consultez', 'attendez', 'demandez', 'utilisez', 'modifiez' /* etc. — copy depuis UxConventionsTest */];

    public function testNewCodesExist(): void
    {
        foreach (self::NEW_CODES as $code) {
            $this->assertTrue(ErrorDictionary::hasMessage($code), "Code {$code} missing from ErrorDictionary");
        }
    }

    public function testNewCodesConformErr02Err03(): void
    {
        foreach (self::NEW_CODES as $code) {
            $msg = ErrorDictionary::getMessage($code);
            // ERR-02 : virgule + verbe action
            $this->assertStringContainsString(',', $msg, "Code {$code} missing comma");
            $hasVerb = false;
            foreach (self::ACTION_VERBS as $verb) {
                if (mb_stripos($msg, $verb) !== false) { $hasVerb = true; break; }
            }
            $this->assertTrue($hasVerb, "Code {$code} missing action verb after comma");
            // ERR-03 : 0 phrase bannie
            foreach (self::FORBIDDEN_PATTERNS as $pattern) {
                $this->assertSame(0, preg_match($pattern, $msg), "Code {$code} contains forbidden phrase");
            }
        }
    }

    public function testBusinessErrorCallersBaseline(): void
    {
        // Find new business_error callers in production code
        $output = shell_exec("grep -rEc \"api_fail\\(\\s*'business_error'\" app/ public/api/ 2>/dev/null | grep -v ':0' | wc -l");
        $count = (int) trim($output ?? '0');
        $this->assertLessThanOrEqual(3, $count, 'Too many business_error callers remaining (expected ≤3 after migration)');
    }
}
```

Tester : `timeout 60 php vendor/bin/phpunit tests/Security/ErrorDictionaryMigrationTest.php --no-coverage`. 3/3 GREEN attendu.
