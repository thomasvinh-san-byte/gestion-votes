Class structure — two public methods:

```php
namespace AgVote\Service;

use Dompdf\Dompdf;
use Dompdf\Options;

final class ProcurationPdfService {
    public function renderHtml(array $proxy, array $meeting, string $orgName): string { ... }
    public function generatePdf(array $proxy, array $meeting, string $orgName): string { ... }
}
```

`renderHtml` builds a self-contained HTML string with inline styles (Dompdf cannot fetch external CSS). Structure:

- `<!DOCTYPE html><html lang="fr"><head><meta charset="UTF-8"><style>...</style></head><body>`
- CSS: `@page { margin: 2cm; }`, body DejaVu Sans 12pt, h1 centered large, `.label` bold, `.value` normal, `.legal-box` border dashed, footer small grey centered.
- **Header block**: organization name (`$orgName`) centered bold, then "POUVOIR DE REPRESENTATION" as h1.
- **Identity block** (table 2 rows):
  - Row 1: "Je soussigne(e) :" | `$proxy['giver_name']`
  - Row 2: "Donne procuration a :" | `$proxy['receiver_name']`
- **Session block**:
  - "Pour la seance :" | `$meeting['title']`
  - "Date :" | formatted date from `$meeting['scheduled_at']` using `date('d/m/Y', strtotime(...))`, or "—" if null
  - "Organisation :" | `$orgName`
- **Legal mention box** (`.legal-box`):
  ```
  Conformement aux dispositions legales applicables aux associations et collectivites,
  le present pouvoir autorise le mandataire a voter au nom du mandant pour toutes
  les resolutions inscrites a l'ordre du jour de la seance susmentionnee.
  ```
- **Signature block** (two columns side by side using a table):
  - Left: "Signature du mandant :" + blank line
  - Right: "Signature du mandataire :" + blank line
- **Footer**: "Document genere par AG-VOTE le " + `date('d/m/Y a H:i')` + " — Identifiant : " + `$proxy['id']`
- Close `</body></html>`

`generatePdf` calls `renderHtml`, then instantiates Dompdf with `isHtml5ParserEnabled=true`, `isRemoteEnabled=false`, `defaultFont=DejaVu Sans`, loads HTML UTF-8, sets paper A4 portrait, renders, and returns `$dompdf->output()`.

Create `app/Controller/ProcurationPdfController.php` in namespace `AgVote\Controller`.

```php
namespace AgVote\Controller;

use AgVote\Service\ProcurationPdfService;

final class ProcurationPdfController extends AbstractController {
    public function download(): void { ... }
}
```

`download()` method body:
1. Read params: `$proxyId = api_query('proxy_id')`, `$meetingId = api_query('meeting_id')`.
2. Validate both are non-empty UUIDs; call `api_fail('missing_proxy_id', 400)` / `api_fail('invalid_proxy_id', 400)` / same for meeting_id.
3. `$tenantId = api_current_tenant_id()`.
4. Load meeting: `$meeting = $this->repo()->meeting()->findByIdForTenant($meetingId, $tenantId)`. If null or `$meeting['tenant_id'] !== $tenantId` → `api_fail('meeting_not_found', 404)`.
5. Load proxy: `$proxy = $this->repo()->proxy()->findWithNames($proxyId, $meetingId, $tenantId)`. If null → `api_fail('proxy_not_found', 404)`.
6. Load org name: `$orgName = $this->repo()->settings()->get($tenantId, 'org_name') ?? ''`.
7. Generate PDF: `$pdf = (new ProcurationPdfService())->generatePdf($proxy, $meeting, $orgName)`.
8. Build filename: `'PROCURATION_' . preg_replace('/[^a-zA-Z0-9_-]/', '_', $proxy['giver_name'] ?? 'mandant') . '_' . date('Ymd') . '.pdf'`.
9. Emit headers and body:
   ```php
   header('Content-Type: application/pdf');
   header('Content-Disposition: attachment; filename="' . $filename . '"');
   header('Content-Length: ' . strlen($pdf));
   header('X-Content-Type-Options: nosniff');
   echo $pdf;
   ```
10. Audit log: `audit_log('procuration.pdf_download', 'proxy', $proxyId, ['giver_name' => $proxy['giver_name'], 'receiver_name' => $proxy['receiver_name']], $meetingId)`.
