**Systematic pass — for each of the remaining pages:**

Go through settings.htmx.html, wizard.htmx.html, archives.htmx.html, members.htmx.html, meetings.htmx.html, users.htmx.html, and audit.htmx.html. For EACH interactive element that lacks an accessible name, add one using these rules:
1. `<button>` with visible text: no change needed (text IS the accessible name)
2. `<button>` with ONLY svg/icon and no text: add `aria-label="[French action description]"`
3. `<input>` with associated `<label for="id">`: no change needed
4. `<input>` with no id and no label: add `aria-label="[field description]"`
5. `<select>` with `<label for="id">`: no change needed
6. `<a>` with visible text: no change needed
7. `<a>` with only icon, no text, no aria-label: add `aria-label="[destination or action]"`
8. Buttons with only `title` attribute and icon: add `aria-label` matching the title text
