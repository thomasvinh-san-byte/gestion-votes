<objective>
Make the official PV PDF immutable after first generation for a validated meeting.

Purpose: QUAL-01 requires that once a session is validated, the PV cannot be regenerated differently. Currently generatePdf() re-queries the database and re-runs Dompdf on every call, so data changes after validation could alter the PDF. Fix: on first call for a validated meeting, generate and store the HTML in meeting_reports via upsertFull() (already done). On subsequent calls, reconstruct the PDF from the stored HTML snapshot instead of re-querying.

Output: generatePdf() checks meeting_reports.html snapshot before generating. If snapshot exists and !$isPreview, build PDF from stored HTML. If no snapshot, generate as before and store. Preview calls always regenerate fresh.
</objective>

After the existing block that validates meeting existence and validated_at check (around line 351, after the `api_fail('meeting_not_validated', 409, ...)` block), add:

```php
        // QUAL-01: Serve immutable snapshot for validated meetings.
        // If a snapshot HTML exists for this meeting, rebuild the PDF from it
        // without re-querying the database — ensuring immutability post-validation.
        if (!$isPreview) {
            try {
                $snap = $this->repo()->meetingReport()->findSnapshot($meetingId, $tenantId);
                if ($snap && !empty($snap['html'])) {
                    $snapHtml = (string) $snap['html'];
                    $snapOptions = new Options();
                    $snapOptions->set('isHtml5ParserEnabled', true);
                    $snapOptions->set('isRemoteEnabled', false);
                    $snapOptions->set('defaultFont', 'DejaVu Sans');
                    $snapDompdf = new Dompdf($snapOptions);
                    $snapDompdf->loadHtml($snapHtml, 'UTF-8');
                    $snapDompdf->setPaper('A4', 'portrait');
                    $snapDompdf->render();
                    $snapPdf = $snapDompdf->output();
                    $snapHash = hash('sha256', $snapPdf);

                    $safeTitle = preg_replace('/[^a-zA-Z0-9_-]/', '_', $meeting['title'] ?? 'seance');
                    $filename = 'PV_' . $safeTitle . '_' . date('Ymd') . '.pdf';
                    $disposition = $isInline ? 'inline' : 'attachment';

                    header('Content-Type: application/pdf');
                    header('Content-Disposition: ' . $disposition . '; filename="' . $filename . '"');
                    header('Content-Length: ' . strlen($snapPdf));
                    header('X-Content-Type-Options: nosniff');
                    header('X-Report-SHA256: ' . $snapHash);
                    header('X-PV-Snapshot: true');
                    echo $snapPdf;
                    return;
                }
            } catch (Throwable) {
                // Snapshot read failed — fall through to fresh generation
            }
        }
```
