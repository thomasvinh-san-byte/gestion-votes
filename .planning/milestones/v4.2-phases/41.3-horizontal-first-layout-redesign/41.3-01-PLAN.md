  <action>
Add `.form-grid-3` utility immediately after the `.form-grid-2` block (after the `@media (max-width: 480px)` closing brace for form-grid-2, before the `/* --- Tables --- */` comment).

Exact CSS to add:

```css
.form-grid-3 {
  display: grid;
  grid-template-columns: repeat(3, 1fr);
  gap: var(--space-field) 12px;
}

.form-grid-3 .form-group-full {
  grid-column: 1 / -1;
}

@media (max-width: 1024px) {
  .form-grid-3 { grid-template-columns: 1fr 1fr; }
}

@media (max-width: 768px) {
  .form-grid-3 { grid-template-columns: 1fr; }
}
```

This mirrors the `.form-grid-2` pattern with a 3-step breakpoint cascade: 3col (1366px+) -> 2col (1024px) -> 1col (768px). The intermediate 2-column step at 1024px gives smoother degradation than jumping directly to 1-column (Pitfall 6 from RESEARCH.md).
  </action>

- Collapses to 2 columns at max-width 1024px
- Collapses to 1 column at max-width 768px
- gap uses `var(--space-field) 12px` (same as .form-grid-2)
