# Phase 41.5: Analytics Page Deep Redesign - Context

**Gathered:** 2026-03-20
**Status:** Ready for planning

<domain>
## Phase Boundary

Deep visual redesign of analytics.htmx.html — the statistics/dashboard page. KPIs, chart layout, filter bar, data tables, donut chart, anomaly indicators all need modernization. Horizontal-first, modern dashboard, Stripe/Linear analytics quality.

</domain>

<decisions>
## Implementation Decisions

### KPI Cards
- **Horizontal layout** with icon left, value + label + trend right (same pattern as admin Phase 41.4)
- **Trend indicators** more prominent — colored arrow + percentage change, not just tiny text
- **Context line** below value ("vs période précédente", "+12% ce mois")
- Full width horizontal row — use available space

### Chart Layout
- **Featured chart** pattern — one "hero" chart at larger size, others in 2-col grid below
- **Participation tab:** Hero line chart full width, detail table below
- **Résolutions tab:** Donut chart larger with legend beside it (horizontal), not below
- **Chart cards:** Proper card treatment — title (semibold), subtitle (muted), chart canvas with generous padding
- **Chart hover:** Info button and export button visible on card hover (already done Phase 38 — verify)
- **3-column option** at 1400px+ for smaller charts

### Filter Bar
- **Period pills** larger and more prominent — 36px height, clear active state with primary fill
- **Refresh button** with icon + text, not just icon
- **Visual grouping** — period pills as a segmented control (connected pills) rather than loose buttons
- **Sticky** — filter bar sticks below header on scroll

### Donut Chart
- **Larger** — 240px diameter on desktop (up from 180px)
- **Legend horizontal** beside the donut on desktop, below on mobile
- **Interactive** — hover segments to highlight + show value
- **Center text** prominent — total votes in JetBrains Mono large

### Data Tables
- **Alternating rows** — subtle stripe for scanability
- **Sticky headers** (already from design system)
- **Numeric columns** in JetBrains Mono
- **Horizontal density** — use available width

### Anomaly Indicators
- **Card treatment** instead of list items — each anomaly in a mini-card
- **Status icons** (warning triangle, info circle, checkmark) instead of just colored borders
- **2-column grid** on desktop for anomaly items

### Tab Navigation
- **Modern underline** animation (sliding indicator)
- **Tab badge counts** showing number of items per tab

### Claude's Discretion
- Which chart is the "hero" featured chart
- Exact donut SVG sizing and legend layout
- Whether to add sparklines to KPI cards
- Anomaly card exact layout proportions
- Tab animation implementation

</decisions>
