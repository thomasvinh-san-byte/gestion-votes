## Decisions Made

- **Override `.op-sidebar` max-height**: The existing `max-height: calc(100vh - 280px)` was a fragile magic number. Scoped override `.operator-right.op-sidebar { max-height: none; flex: 1; overflow-y: auto; }` replaces it with proper flex fill behavior.
