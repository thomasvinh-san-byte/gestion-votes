- Modal open animation now uses bouncy `cubic-bezier(.34,1.2,.64,1)` easing — tactile spring feel matching wireframe
- Toast container moved to bottom-right with `column-reverse` stacking so newest toasts anchor at bottom
- Complete toast sub-element API: `.toast-icon`, `.toast-title`, `.toast-msg`, `.toast-close` with hover states
- `@keyframes toastOut` exit animation (opacity 0 + translateX 100%) with `.toast-exit` utility class
- Interactive row arrow glows from `opacity: .3` to `opacity: 1` and slides `translateX(2px)` on parent hover
- `.irow:hover .irow-title` accents to `var(--color-primary)` for clear navigation affordance
- `pageIn` animation tightened to `translateY(4px) / 0.18s` (was `6px / 0.25s`)
