**Method 2: `scheduleForMembers(...): array`** — Unify the shared loop from scheduleInvitations (lines 148-246), scheduleReminders (lines 274-342), and scheduleResults (lines 376-444). Parameters:
```php
public function scheduleForMembers(
    string $tenantId,
    string $meetingId,
    ?string $templateId,
    ?string $scheduledAt,
    string $defaultSubject,
    string $defaultBodyConstant,
    bool $checkUnsent = false,
    bool $createInvitation = false,
): array
```
The unified loop:
1. Init `$result = ['scheduled'=>0, 'skipped'=>0, 'errors'=>[]]`, `$offset=0`, `$batchSize=25`
2. Paginated member loop via `$this->memberRepo->listActiveWithEmailPaginated()`
3. Skip: wrong tenant_id, empty email (increment `$result['skipped']`)
4. If `$checkUnsent`: check `$this->invitationRepo->findStatusByMeetingAndMember()`, skip if 'sent'
5. If `$createInvitation`: generate token `bin2hex(random_bytes(16))`, call `$this->invitationRepo->upsertBulk()`, retrieve invitation via `findByMeetingAndMember()`. Otherwise: token = `''`, invitationId = `null`.
6. Render template: if `$templateId` -> try `renderTemplate()`, on failure fallback to `getVariables() + renderHtml($defaultSubject, $variables) + renderHtml($defaultBodyConstant, $variables)`. If no templateId -> same fallback path directly.
7. Enqueue via `$this->queueRepo->enqueue()` with all params including `$invitationId`, `$member['full_name']`
8. On success: `$this->eventRepo->logEvent()` with 'queued', increment scheduled. On failure: push error.
9. Continue pagination until `count($members) < $batchSize`.

**Method 3: `sendImmediate(string $tenantId, string $meetingId, ?string $templateId, bool $onlyUnsent, int $limit): array`** — Extract lines 486-518 from sendInvitationsNow. Includes the isConfigured guard returning early error. Delegates to `sendImmediateBatch()` for actual member processing.

**Method 4: `sendImmediateBatch(array $members, string $tenantId, string $meetingId, ?string $templateId, bool $onlyUnsent, array &$result): void`** — Extract lines 524-603 from sendInvitationsNowBatch verbatim. Private method. Member filtering, template rendering, mailer->send(), invitation upsert on success, markBounced on failure.
