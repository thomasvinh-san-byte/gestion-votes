### Task 3: Update JS to render table rows with avatar and pagination

1. **Avatar generation**: Extract initials from user name, generate consistent color from name hash:
```javascript
function getInitials(name) {
  var parts = name.trim().split(/\s+/);
  if (parts.length >= 2) return (parts[0][0] + parts[parts.length - 1][0]).toUpperCase();
  return name.substring(0, 2).toUpperCase();
}
function getAvatarColor(name) {
  var colors = ['#1650E0','#059669','#D97706','#DC2626','#7C3AED','#0891B2','#BE185D','#4F46E5'];
  var hash = 0;
  for (var i = 0; i < name.length; i++) hash = name.charCodeAt(i) + ((hash << 5) - hash);
  return colors[Math.abs(hash) % colors.length];
}
```

### Task 4: Format last_login date
**File**: `public/assets/js/pages/admin.js`

Add date formatting for last_login field. If the API returns ISO date, format as "dd/mm/yyyy HH:mm". If null/undefined, show "Jamais" in muted text.

```javascript
function formatLastLogin(dateStr) {
  if (!dateStr) return '<span class="text-muted">Jamais</span>';
  var d = new Date(dateStr);
  return d.toLocaleDateString('fr-FR') + ' ' + d.toLocaleTimeString('fr-FR', {hour:'2-digit', minute:'2-digit'});
}
```
