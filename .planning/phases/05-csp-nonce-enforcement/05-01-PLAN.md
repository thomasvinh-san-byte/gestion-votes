**3. Add nonce to all <script src="..."> tags in .htmx.html files:**

Per `strict-dynamic` CSP spec: when `strict-dynamic` is present, browsers ignore `'self'` in script-src. Every `<script src="...">` tag MUST carry the nonce. This is critical.

For all 21 .htmx.html files, add `nonce="%%CSP_NONCE%%"` to every `<script>` tag (both `src=` and inline). Use sed:
```bash
sed -i 's/<script src="/<script nonce="%%CSP_NONCE%%" src="/g' public/*.htmx.html
```

Also handle `<script type="module" src="...">` patterns:
```bash
sed -i 's/<script type="module" src="/<script type="module" nonce="%%CSP_NONCE%%" src="/g' public/*.htmx.html
```
