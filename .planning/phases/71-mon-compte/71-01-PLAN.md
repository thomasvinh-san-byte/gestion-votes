    Create following exact pattern of PasswordResetRedirectException. Namespace AgVote\Controller. Extends \RuntimeException. Constructor takes string $location, int $httpCode. Properties: public string $location, public int $httpCode.

    **AccountController** (app/Controller/AccountController.php):
    - Namespace AgVote\Controller. Does NOT extend AbstractController. Uses HtmlView::render().
    - Constructor: `__construct(?UserRepository $userRepo = null)` — default via RepositoryFactory::getInstance()->user()
    - Main method `account(): void` dispatched by router:
      1. Start session via SessionHelper::start()
      2. Check `$_SESSION['auth_user']` — if empty, redirect to `/login?redirect=/account`
      3. Read user from session: `$user = $_SESSION['auth_user']` (keys: id, tenant_id, email, name, role)
      4. If GET: render account_form with `['user' => $user, 'errors' => [], 'success' => false]`
      5. If POST: call handlePasswordChange($user)
    - Private `handlePasswordChange(array $sessionUser): void`:
      1. Read POST fields: current_password, new_password, new_password_confirm
      2. Validate: current_password required, new_password min 8 chars, new_password === new_password_confirm
      3. If validation errors, re-render account_form with errors array and user data
      4. Load fresh user from DB: `$this->userRepo->findByEmailGlobal($sessionUser['email'])`
      5. If user not found or not active, redirect to /login
      6. Verify current password: `password_verify($currentPassword, $dbUser['password_hash'])`
      7. If wrong: re-render with error `['Le mot de passe actuel est incorrect.']`
      8. Hash new password: `password_hash($newPassword, PASSWORD_DEFAULT)`
      9. Save: `$this->userRepo->setPasswordHash($dbUser['tenant_id'], $dbUser['id'], $hash)`
      10. Audit: `audit_log('password_changed', 'user', $dbUser['id'], ['ip' => $_SERVER['REMOTE_ADDR'] ?? 'unknown'])`
      11. Re-render account_form with `['user' => $sessionUser, 'errors' => [], 'success' => true]`
    - Private `redirect(string $location, int $code = 302): never` — same pattern as PasswordResetController (throws AccountRedirectException if PHPUNIT_RUNNING)
