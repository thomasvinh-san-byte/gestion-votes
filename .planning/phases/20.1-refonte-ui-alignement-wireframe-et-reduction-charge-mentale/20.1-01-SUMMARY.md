## What was built

Refonte complete de la page votant tablette alignee sur le wireframe v3.19.2:

1. **Grille 2x2** — Vote buttons now display in a 2-column grid (Pour/Contre top row, Abstention/Blanc bottom row) with max-width: 600px, replacing the previous 4-column layout.

2. **Confirmation inline coloree** — Selecting a vote choice shows a full-width colored confirm button below the grid (green for Pour, red for Contre, gray for Abstention/Blanc). The old dark overlay bottom sheet is preserved as fallback but never shown.

3. **Header minimal** — Replaced title "Vote" and subtitle with a single resolution counter (Resolution 3/5) + timer + zoom button. Removed refresh button.

4. **FOUC fix** — Inline style id="critical-tokens" block in head provides background and text colors before any stylesheet loads.

5. **Resolution counter** — MutationObserver watches #motionProgressText (set by vote.js) and mirrors it to #voteResolutionCounter in the minimal header.
