**1. Migration** â€” Create `database/migrations/20260401_password_resets.sql`:
```sql
CREATE TABLE IF NOT EXISTS password_resets (
    id UUID PRIMARY KEY DEFAULT gen_random_uuid(),
    tenant_id UUID NOT NULL REFERENCES tenants(id) ON DELETE CASCADE,
    user_id UUID NOT NULL REFERENCES users(id) ON DELETE CASCADE,
    token_hash CHAR(64) NOT NULL,
    expires_at TIMESTAMPTZ NOT NULL,
    used_at TIMESTAMPTZ,
    created_at TIMESTAMPTZ NOT NULL DEFAULT now()
);
CREATE INDEX IF NOT EXISTS idx_password_resets_token_hash ON password_resets(token_hash);
CREATE INDEX IF NOT EXISTS idx_password_resets_user ON password_resets(user_id);
CREATE INDEX IF NOT EXISTS idx_password_resets_expires ON password_resets(expires_at) WHERE used_at IS NULL;
```
Also add this table definition to `database/schema-master.sql` after the users table section (before email_templates).

**2. PasswordResetRepository** â€” Create `app/Repository/PasswordResetRepository.php`:
- Extends AbstractRepository
- `insert(string $tokenHash, string $tenantId, string $userId, string $expiresAt): void` â€” INSERT INTO password_resets
- `findByHash(string $tokenHash): ?array` â€” SELECT * WHERE token_hash = :hash AND used_at IS NULL AND expires_at > NOW() LIMIT 1
- `markUsed(string $tokenHash): int` â€” UPDATE SET used_at = NOW() WHERE token_hash = :hash AND used_at IS NULL, return affected rows
- `deleteExpired(): int` â€” DELETE FROM password_resets WHERE expires_at < NOW() - INTERVAL '24 hours' (cleanup)
- `deleteForUser(string $userId): int` â€” DELETE FROM password_resets WHERE user_id = :uid (invalidate old tokens on successful reset)

**4. PasswordResetService** â€” Create `app/Services/PasswordResetService.php`:
- Namespace: `AgVote\Service`
- Constructor accepts optional `?PasswordResetRepository`, `?UserRepository`, `?EmailQueueRepository` (same DI pattern as VoteTokenService)
- Private constant `TOKEN_BYTES = 32`, `TTL_SECONDS = 3600` (1 hour)
- Private static `hashToken(string $rawToken): string` â€” `hash_hmac('sha256', $rawToken, APP_SECRET)` (same pattern as VoteTokenService)
- `requestReset(string $email): void` â€” NO return value (prevents user enumeration):
  1. Find user via `userRepo->findByEmailGlobal($email)`
  2. If no user found OR user not active: return silently (no error)
  3. Generate token: `bin2hex(random_bytes(self::TOKEN_BYTES))`
  4. Hash: `self::hashToken($rawToken)`
  5. Compute expiresAt: `gmdate('Y-m-d\TH:i:s\Z', time() + self::TTL_SECONDS)`
  6. Store: `resetRepo->insert($tokenHash, $user['tenant_id'], $user['id'], $expiresAt)`
  7. Build reset URL: `$baseUrl = rtrim((string)(getenv('APP_URL') ?: 'http://localhost:8080'), '/'); $resetUrl = $baseUrl . '/reset-password?token=' . $rawToken;`
  8. Build HTML email body with French text: "Bonjour {name}, Vous avez demande la reinitialisation de votre mot de passe. Cliquez sur le lien ci-dessous (valable 1 heure) : [link]. Si vous n'etes pas a l'origine de cette demande, ignorez cet email."
  9. Queue via `emailQueueRepo->enqueue($user['tenant_id'], $user['email'], 'Reinitialisation de votre mot de passe â€” AG-VOTE', $html)`
- `validateToken(string $rawToken): ?array` â€” hash, findByHash, return row or null
- `resetPassword(string $rawToken, string $newPassword): array` â€” returns `['ok' => bool, 'error' => ?string]`:
  1. Hash and find token via findByHash
  2. If not found: return `['ok' => false, 'error' => 'token_invalid']`
  3. Hash new password: `password_hash($newPassword, PASSWORD_DEFAULT)`
  4. Update user: `userRepo->setPasswordHash($row['tenant_id'], $row['user_id'], $hash)`
  5. Mark token used: `resetRepo->markUsed($tokenHash)`
  6. Delete all other tokens for this user: `resetRepo->deleteForUser($row['user_id'])`
  7. Audit log: `audit_log('password_reset', 'user', $row['user_id'], ['ip' => $_SERVER['REMOTE_ADDR'] ?? 'unknown'])`
  8. Return `['ok' => true, 'error' => null]`

- Main method `resetPassword(): void`:
  - Check for `?token=` query param:
    - If token present AND method is GET: call `validateToken()`, if valid render `reset_newpassword_form` template, if invalid render `reset_request_form` with error "Ce lien de reinitialisation est invalide ou a expire."
