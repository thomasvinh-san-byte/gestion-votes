## Decisions Made

- **Token `--radius-pill` absent** : design-system.css ne définit pas `--radius-pill` (uniquement utilisé en fallback `var(--radius-pill, 9999px)` dans 1 endroit du codebase). J'ai utilisé `--radius-full` (= 9999px) directement, qui est le token consacré.
- **Collapse au breakpoint 1024px** : sidebar passe sous le contenu plutôt qu'à droite. Cohérent avec EDITORIAL-08 (grid mandatoire ≥1024px).
- **Box-shadow xs ajouté sur le pill** : non explicite dans le plan, mais cohérent avec la consolidation Schoger S-2 (TECH-01) — élévation discrète pour différencier le pill du flux. Suppression possible si contre-indiquée par UX review.
- **Sélecteur `details > summary`** : ajouté dans la duality (le plan listait `.btn / button / input / select / .filter-tab / .audit-filter-tabs`). Couvre le pattern HTMX accordéon utilisé sur les pages audit/trust.
