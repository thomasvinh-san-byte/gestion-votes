- `.audit-table`: full-width table with token-based borders, padding, header bg
- `.audit-table th`: sticky top, bg-subtle, text-xs uppercase, text-muted
- `.audit-table td`: text-sm, border-bottom
- `.audit-table tr:hover td`: bg-subtle
- `.audit-table .audit-checkbox`: width 16px checkbox column
- `.audit-timeline`: vertical layout with left time markers and right cards
- `.audit-timeline-item`: flex row, relative positioning for connector line
- `.audit-timeline-connector`: 2px vertical line using --color-border
- `.audit-timeline-card`: card-style with surface bg, border, radius, padding
- `.audit-fingerprint`: bg-subtle box with mono font, border, radius
- `.audit-hash`: font-mono, text-xs, word-break, user-select-all
- Responsive: stack filters vertically on mobile, hide some table columns on < 768px
- Use `var(--color-*)`, `var(--space-*)`, `var(--text-*)`, `var(--radius-*)` tokens throughout — zero hardcoded values
