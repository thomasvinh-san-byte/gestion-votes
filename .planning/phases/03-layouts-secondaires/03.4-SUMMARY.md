# Phase 3 Plan 4 : Simplification page login — Summary

**One-liner :** Suppression du pattern `login-brand-grid` (HTML+CSS) et de l'orbe animé `login-orb` (CSS), réduction du panneau brand à 1 seul bénéfice (Vote sécurisé), glow conservé atténué — pour donner plus de présence visuelle au formulaire.

## What was built

### LOGIN-01 — Suppression pattern + orbe

Supprimé du HTML (`public/login.html`) :
- `<div class="login-brand-grid" aria-hidden="true">` (anciennement ligne 28)

Supprimé du CSS (`public/assets/css/login.css`) :
- Toutes règles `.login-brand-grid` (background-image gradient pattern + positionnement)
- Toutes règles `.login-orb` (radial-gradient animé)
- `@keyframes brand-glow-drift` (devenue orpheline après suppression)
- Règle `prefers-reduced-motion` qui désactivait l'animation drift (devenue obsolete)

Conservé :
- `.login-brand-glow` (single subtle gradient layer ≠ pattern répétitif) — atténué :
  - `opacity` : 0.18 → 0.12
  - `animation: brand-glow-drift ...` : retiré
  - `filter: blur(20px)` conservé
  - Commentaire inline ajouté dans le CSS pour documenter l'atténuation

### LOGIN-02 — Réduction features 3 → 1

`<ul class="login-brand-features">` passe de 3 `<li>` à 1 seul :
- Supprimé : `<li>` "Quorum paramétrable" (1er)
- **Conservé** : `<li>` "Vote sécurisé — Token unique, chiffrement, anti-rejeu" (2e)
- Supprimé : `<li>` "PV signé et figé" (3e)

Libellé du `<li>` Vote sécurisé conservé verbatim sans modification.

### F4 fix — ratio form-dominant verrouillé

`grid-template-columns: 2fr 3fr` (40% brand / 60% form) maintenu. La baseline n'avait pas dérivé vers `3fr 2fr` ; verrouillage par construction.
