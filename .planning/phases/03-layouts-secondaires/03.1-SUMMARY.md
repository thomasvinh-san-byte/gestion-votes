## Deviations from Plan

### Auto-fixed Issues

**1. [Rule 3 — Blocking] Mise à jour de dashboard.js pour utiliser le nouveau empty state**
- **Found during:** Task 3 (empty state).
- **Issue:** Le plan demandait d'ajouter le markup `<ag-empty-state icon="none">` dans `dashboard.htmx.html`, mais le rendu de l'empty state était déjà inlined dans `dashboard.js` ligne 150 avec un texte différent (`icon="meetings"`, `title="Aucune séance"`).
