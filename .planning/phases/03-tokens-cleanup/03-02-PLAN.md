    **Étape A — Recompter les tokens 1-site post-Phase 7.4 :**

    Pour chaque token `--border-*` ou `--shadow-*` défini dans `design-system.css`, compter ses callers hors-design-system.css :

    ```bash
    # Lister tous les tokens définis (--border-* et --shadow-*)
    grep -oE "^\s*--(border|shadow)-[a-z0-9-]+:" public/assets/css/design-system.css | sed 's/^\s*//;s/:$//' | sort -u > /tmp/tokens.txt

    # Pour chaque token, compter les callers hors design-system.css
    while read token; do
      hits=$(grep -rE "var\\(${token}\\)" public/assets/css/ public/*.html app/ 2>/dev/null | grep -v "design-system.css:" | wc -l)
      echo "$hits $token"
    done < /tmp/tokens.txt | sort -n
    ```

    Compter les tokens avec exactement **1 caller** = décompte tokens 1-site final.
    **Cible attendue : <30** (objectif TOKENS-V26-04). L'audit v2.5 prévoyait 29.

    **Étape B — Mesurer le ratio tokens borders/shadows :**

    ```bash
    # Bordures hardcoded vs tokens
    HARDCODED_BORDER=$(grep -rEc "^\s*border(-[a-z]+)?:\s+[0-9]+px" public/assets/css/ | grep -v "design-system.css" | awk -F: '{s+=$2}END{print s}')
    TOKEN_BORDER=$(grep -rEc "border:\s+var\\(--border" public/assets/css/ | grep -v "design-system.css" | awk -F: '{s+=$2}END{print s}')
    # Idem pour box-shadow vs var(--shadow-*)
    HARDCODED_SHADOW=$(grep -rEc "^\s*box-shadow:\s+[0-9]" public/assets/css/ | grep -v "design-system.css" | awk -F: '{s+=$2}END{print s}')
    TOKEN_SHADOW=$(grep -rEc "box-shadow:\s+var\\(--shadow" public/assets/css/ | grep -v "design-system.css" | awk -F: '{s+=$2}END{print s}')
    # Ratio = TOKEN / (TOKEN + HARDCODED) * 100, doit être ≥95 %
    ```
