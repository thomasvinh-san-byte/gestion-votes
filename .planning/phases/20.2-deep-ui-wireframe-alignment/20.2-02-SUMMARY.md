## Accomplishments

- Form inputs now use 1px border (down from 2px), surface-raised background, and 42px min-height; small variant gets 34px
- Form labels are 13px/700/uppercase with 0.5px letter-spacing — matches wireframe v3.19.2 data-entry spec
- Tables get sticky uppercase headers with 0.7px letter-spacing, per-td overflow ellipsis at 260px max-width, striped even rows, and accent-tinted hover using color-mix
- Tags changed from pill (radius-full) to 5px rounded rectangle; all semantic variants (.tag-accent/danger/success/warn/purple) gain subtle colored borders via color-mix

## Decisions Made

- Tag hover selector changed to `tr:hover td` (not `tr:hover`) so it properly overrides the per-td striped background applied via nth-child(even). Without this, the stripe would win over hover on even rows.
- `color-mix()` chosen for semantic tag borders — produces a 30%-saturated version of each semantic color without adding new CSS custom properties.
