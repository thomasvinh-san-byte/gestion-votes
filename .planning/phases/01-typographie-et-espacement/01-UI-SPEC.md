## Typography

### Token Changes (Phase 1 scope)

| Token | Current | Target | Requirement |
|-------|---------|--------|-------------|
| `--text-base` | 0.875rem (14px) | 1rem (16px) | TYPO-01 |
| `--type-label-size` | `var(--text-sm)` = 13px | `var(--text-base)` = 16px (after change) | TYPO-02 |
| `.form-label` text-transform | `uppercase` | `none` | TYPO-02 |
| `.form-label` color | `var(--color-text-muted)` | `var(--color-text-dark)` | TYPO-02 |
| `.form-label` letter-spacing | `0.5px` | `0` | TYPO-02 |
| `.nav-group-label` font-size | `12px` (hardcoded) | `var(--text-xs)` = 12px (tokenized, no visual change) | — |
| `body` line-height | `var(--leading-base)` = 1.571 | Update to `var(--leading-md)` = 1.5 | TYPO-01 (16px × 1.5 = 24px) |

### Declared Type Scale (post-change state)

| Role | Size | CSS Token | Weight | Line Height | Font |
|------|------|-----------|--------|-------------|------|
| Body | 16px | `--text-base` (updated) | 400 regular | 1.5 | Bricolage Grotesque |
| Label | 16px | `--type-label-size` → `--text-base` | 600 semibold | 1.55 | Bricolage Grotesque |
| Section heading | 24px | `--text-2xl` | 600 semibold | 1.3 | Bricolage Grotesque |
| Page title | 30px | `--text-3xl` | 600 semibold | 1.2 | Fraunces (display) |

**Weights used (2 max):** 400 regular + 600 semibold. No exceptions.

**Header composition (TYPO-03):**
- Height: 64px (token `--header-height: 64px`)
- Content: breadcrumb (--text-sm = 13px, weight 400, color `--color-text-muted`) + page title (--text-3xl, Fraunces, weight 600 semibold)
- Removed: subtitle text, decorative bottom bar
- Action buttons: moved to toolbar below header, not inside header

Source for current values: `design-system.css` lines 196–216, 237–240, 577–614, 1813–1819.

| Role | CSS Token | Light Mode Value | Usage |
|------|-----------|-----------------|-------|

- `.form-label` color changes from `--color-text-muted` (oklch(0.340 0.030 257)) to `--color-text-dark` (oklch(0.208 0.042 265) in light mode). Source: CONTEXT.md decision, TYPO-02.

Accent reserved for: primary CTA buttons, active sidebar nav item indicator, focus ring on interactive elements, interactive link hover underlines. NOT used for decorative elements or section borders.

## Copywriting Contract

Phase 1 is a pure token/CSS change phase — no new UI screens, no new interactive flows. Copywriting contract covers the header composition change only.

| Element | Copy |
|---------|------|
| Primary CTA | Not applicable — no new CTA introduced in this phase |
| Header breadcrumb separator | `/` (forward slash, plain text, not an icon) |
| Empty state heading | Not applicable — no empty states introduced in this phase |
| Error state | Not applicable — no new error flows in this phase |
| Destructive confirmation | Not applicable — no destructive actions in this phase |

**Labels — TYPO-02 contract:**
- Form labels display in sentence case (e.g., "Nom du membre", not "NOM DU MEMBRE")
- No letter-spacing offset

## Interaction Contract

Phase 1 has no new interactions. Existing interactions inherit token updates automatically via CSS cascade.

| Component | Behavior | Change from current |
|-----------|----------|---------------------|
| All text nodes | Render at 16px base instead of 14px | Font size increase via `--text-base` token |
| Form labels | Normal case, dark color, no uppercase | CSS rule changes on `.form-label` |
| Page header | 64px height, breadcrumb + title only | `--header-height` token + header HTML trim |
| Form fields | 20px gap between fields | `--form-gap` alias applied to form stacks |
| Sections | 24px gap between sections | `--section-gap` alias applied to `.section` wrappers |
| Dark theme | Inherits all token changes | No additional work — tokens are theme-agnostic for sizing |
