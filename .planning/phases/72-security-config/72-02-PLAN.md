    **AuthMiddleware changes:**
    1. Keep `private const SESSION_TIMEOUT = 1800;` as the DEFAULT (rename to `DEFAULT_SESSION_TIMEOUT`)
    2. Add a static cache: `private static ?int $cachedSessionTimeout = null;` and `private static ?string $cachedTimeoutTenantId = null;`
    3. Add public static method:
       ```php
       public static function getSessionTimeout(?string $tenantId = null): int
       {
           $tid = $tenantId ?? self::getCurrentTenantId();
           // Cache: avoid DB read on every request
           if (self::$cachedSessionTimeout !== null && self::$cachedTimeoutTenantId === $tid) {
               return self::$cachedSessionTimeout;
           }
           try {
               $repo = RepositoryFactory::getInstance()->settings();
               $val = $repo->get($tid, 'settSessionTimeout');
               if ($val !== null && is_numeric($val)) {
                   $seconds = ((int) $val) * 60; // stored as minutes, used as seconds
                   $seconds = max(300, min(28800, $seconds)); // 5min - 480min
                   self::$cachedSessionTimeout = $seconds;
                   self::$cachedTimeoutTenantId = $tid;
                   return $seconds;
               }
           } catch (\Throwable $e) {
               // DB failure: fall back to default
           }
           self::$cachedSessionTimeout = self::DEFAULT_SESSION_TIMEOUT;
           self::$cachedTimeoutTenantId = $tid;
           return self::DEFAULT_SESSION_TIMEOUT;
       }
       ```
    4. In `authenticate()`, replace `self::SESSION_TIMEOUT` with `self::getSessionTimeout()` (line ~300)
    5. In `reset()`, add: `self::$cachedSessionTimeout = null; self::$cachedTimeoutTenantId = null;`
