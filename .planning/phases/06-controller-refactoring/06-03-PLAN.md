**Slim OperatorController.php:**

```php
public function workflowState(): void {
    $meetingId = api_require_uuid('meeting_id');
    $tenantId = api_current_tenant_id();

    $service = new OperatorWorkflowService($this->repo());
    $result = $service->getWorkflowState($meetingId, $tenantId);

    api_ok($result);
}
```

Apply to all 3 methods: workflowState, openVote, anomalies. Parse HTTP input in controller -> call service -> handle side effects -> api_ok.
