1. **`public function assembleReportHtml(...): string`** — Takes pre-computed data and returns the full HTML report. Parameters: `string $title, string $status, string $president, string $createdAt, string $validatedAt, string $archivedAt, string $rowsHtml, string $attRows, string $attSummary, string $proxyRows, string $proxySummary, string $tokenRows, string $tokenSummary, string $votesByMotionHtml`. 3. **`public function buildMotionRows(array $motions, array $policiesByMotion, array $officialsByMotion): string`** — Moved from private method (lines 486-578). Key change: instead of calling `$this->policyRepo->findVotePolicy()` and `$this->policyRepo->findQuorumPolicy()` per motion, receives pre-fetched maps:
   - `$policiesByMotion[motionId] = ['votePolicy' => ?array, 'quorumPolicy' => ?array]`
   - `$officialsByMotion[motionId] = ['source' => string, 'for' => float, 'against' => float, 'abstain' => float, 'total' => float, 'decision' => string, 'reason' => string, 'note' => string, 'detail' => array]`
   This removes the need for policyRepo, OfficialResultsService, and VoteEngine instantiation inside this method. The method only does HTML row formatting from pre-computed data.

4. **`public function buildAttendanceSection(array $attendance): array`** — Moved as-is (lines 584-603).

5. **`public function buildProxiesSection(array $proxies): array`** — Moved as-is (lines 609-616).

Key change: instead of calling `$this->ballotRepo->listDetailedForMotion()` per motion, receives pre-fetched map `$ballotsByMotion[motionId] = ballots[]`.
