### Task 3 — smtp_configured flag + operator SMTP banner + settings APP_URL warning

**AdminController.php:** Added `smtp_configured` boolean to the `systemStatus()` API response by using `MailerService::buildMailerConfig()` and `isConfigured()`. Added `use AgVote\Service\MailerService` import.

**operator-tabs.js:** Added `checkSmtpWarning()` and `showSmtpWarning()` functions after state variable declarations. `checkSmtpWarning()` is called on page init alongside `loadMeetings()`. It fetches `/api/v1/admin_system_status` and injects a yellow banner if `smtp_configured === false`.

**settings.js:** Added `appUrlLocalhostWarning` logic inside `loadSettings()` `.then()` callback. After populating settings, checks if `app_url` contains `localhost` and `app_env` is not in `['demo', 'development', 'dev', 'local']`. If so, injects an inline yellow warning div after the `app_url` input.
