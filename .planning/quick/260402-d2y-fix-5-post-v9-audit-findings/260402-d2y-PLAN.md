    Finding 3 — N+1 batch groups:
    - Add MemberGroupRepository::listGroupsForMembers(array $memberIds, string $tenantId): array
      Returns an associative array keyed by member_id, each value is the array of groups.
      SQL: single IN() query joining member_group_assignments + member_groups.
    - MembersController::index() when include_groups is set: call listGroupsForMembers()
      with the page's member IDs, then map results onto each member row.
      Remove the per-member foreach loop calling listGroupsForMember().

    Finding 4 — search/filter param:
    - Add MemberRepository::listPaginatedFiltered(string $tenantId, int $limit, int $offset, ?string $search): array
      When $search is non-null and non-empty: add WHERE clause
      `AND (full_name ILIKE :search OR email ILIKE :search)` with :search = '%{$search}%'.
    - Add MemberRepository::countFiltered(string $tenantId, ?string $search): int
      Same filtering logic for the total count.
    - MembersController::index() reads `search` query param (api_query_string or trim($_GET['search']??'')).
      If non-empty, uses listPaginatedFiltered + countFiltered; else uses existing listPaginated + countAll.

    Finding 5 — bulk endpoint:
    - Add MembersController::bulk(): void
      POST /api/v1/members_bulk (route added in Task 1, but add use here too; actually route is in Task 3 — keep routing separate).
      Accepts JSON body:
        { "operation": "assign_group"|"update_voting_power", "member_ids": [...], "group_id"?: "...", "voting_power"?: float }
      Validation:
        - operation required, must be one of assign_group|update_voting_power
        - member_ids required, array, 1..200 items, each a UUID
        - group_id required when operation=assign_group
        - voting_power required when operation=update_voting_power, float 0.01..100
      For assign_group: call MemberGroupRepository::bulkAssignToGroup($groupId, $memberIds)
      For update_voting_power: call MemberRepository::bulkUpdateVotingPower($memberIds, $tenantId, $votingPower)
      Add MemberRepository::bulkUpdateVotingPower(array $ids, string $tenantId, float $power): int
        SQL: UPDATE members SET voting_power = :power, updated_at = NOW()
             WHERE id = IN(:ids) AND tenant_id = :tid AND deleted_at IS NULL
        Use filterExistingIds() to sanitize ids before the UPDATE.
      Response: { "operation": "...", "affected": N }
      Audit log for each operation.
