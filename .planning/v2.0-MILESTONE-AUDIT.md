**1. CRITICAL — Sidebar/mobile Paramètres link stale**
- `partials/sidebar.html` line 109: `href="/admin.htmx.html?tab=settings"` → should be `href="/settings.htmx.html"`
- `shell.js` line 440: mobile bottom nav same broken link
- `data-page="parametres"` in sidebar vs `data-page="settings"` on settings page → no active highlight
- **Fix:** Update sidebar partial + shell.js mobile nav + align data-page values

**2. CRITICAL — users.htmx.html broken script path**
- Line 171: `<script src="/assets/js/pages/meeting-context.js">` → does not exist
- Correct: `/assets/js/services/meeting-context.js`
- **Impact:** MeetingContext undefined on users page; sidebar meeting links silently broken
- **Fix:** One-line path correction
